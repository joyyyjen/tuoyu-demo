課程簡介

FMEA 於 1960 由美國航太工業首先發展，後續導入車界,電子產業與其他工業領域,FMEA 失效模式分析的運用,可減少工程變更與潛 在問題的預防。

課程特色

運用FMEA 軟件,建立有系統的跨功能小組進行產品開發前,生產與 售後服務等減少客戶抱怨的預防活動

課程效益

讓學員了解如何運用 FMEA 手法降低企業品質異常事件的發生率 與危機管理

課程大綱

  1. FMEA 簡介與基本概念,失效模式與關鍵效應分析(FMEA)之意義與重要性
  2. 設計 FMEA 系統方法
  3. 過程(製程)FMEA 系統方法
  4. FMEA 之 RPN 評量準則
  5. FMEA 之完整性（含魚骨圖與失效樹分析－Fault Tree Analysis, FTA 之應用）與效果評價
  6. 新版 VDA-AIAG FMEA Handbook 介紹
  7. FMEA 失效模式分析實施步驟與作業流程
  8. FMEA 與 KM 管理

參加對象 營建,製造與服務業之現場管理幹部,研發,製造,品管,客服與 IE 工程人員
